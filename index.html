Static Relative Absolute Fixed Sticky Z-index Podsumowanie

position: static

  • Domyślne pozycjonowanie
  • Nie działa: top, left, right, bottom
  • Element w normalnym przepływie dokumentu
div {
                position: static;
            }
STATIC

position: relative

  • Względem własnej pierwotnej pozycji
  • Działa: top, left, right, bottom
  • Miejsce w układzie zostaje zachowane
  • Punkt odniesienia dla absolute
div {
            position: relative;
            left: 40px;
        }
RELATIVE

position: absolute

  • Względem najbliższego rodzica z relative
  • Usunięty z normalnego przepływu
  • Może nachodzić na inne elementy
.parent { position: relative; }
    .child {
        position: absolute;
        top: 20px;
        right: 20px;
    }
ABSOLUTE

position: fixed

  • Względem okna przeglądarki
  • Nie przesuwa się przy scrollu
  • Usunięty z przepływu
div {
            position: fixed;
            bottom: 20px;
            right: 20px;
        }
FIXED

position: sticky

  • Łączy relative i fixed
  • Wymaga określenia top
  • Działa w obrębie kontenera
div {
            position: sticky;
            top: 0;
            }
STICKY

z-index

  • Określa kolejność warstw
  • Działa tylko z position ≠ static
  • Większa wartość = wyżej
div {
            position: absolute;
            z-index: 10;
        }
